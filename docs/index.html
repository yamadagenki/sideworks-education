コース紹介

プロのエンジニアを目指す方を対象に，基礎から応用まで3つのコースを設けています．自分の現状に合わせて勉強をしていけるよう教材を揃えています．

Web初級コース

大規模開発を想定したHTML・CSS学習から，Webページが表示されるまでの仕組みなど基礎となる部分を勉強します．基礎的なWebサイトが開発できるようになります．

学習期間：約1ヶ月

Web中級コース

RubyOnRailsを軸に，インフラ構築や，バックエンド開発，Javascript・Sassを用いたフロントエンド開発まで勉強します．標準的なWebサービスが開発できるようになります．

学習期間：約5ヶ月

Web上級コース

指定された要件を満たす，任意のシステムを，企画から開発・運用まで１人でゼロから作成していきます．サービスを開発・運用する上で必要な知識が総合的に得られます．

学習期間：約6ヶ月
